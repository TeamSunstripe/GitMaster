    Git Flow - hotfix

    ここでは、GitFlowにおいて、hotfixブランチ - hotfixについて紹介していきます。

    Git Flow

    hotfix（緊急用）

    hotfixブランチ

    （必須）

    命名規則

    ・hotfix/[バージョン番号]/[バグ識別名]
    ・hotfix/[バグ識別名]

    説明

    • リリースされているデータに、緊急の修正対応で使用
    • 唯一masterブランチに急遽修正を行うときに用いる
